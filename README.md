# Telegram UserBot (WTelegramClient)

## Настройка

1. Получите `api_id` и `api_hash` на [my.telegram.org](https://my.telegram.org)
2. Заполните `config.json`
3. Добавьте пользователей в `allowed_users.txt`
4. Запустите: `dotnet run`

## Безопасность (чтобы не забанили)

- **Задержки**: 2–5 сек между действиями
- **Не более 1 сообщения в 3 сек**
- **Не спамьте в группы**
- **Используйте на запасном аккаунте**
